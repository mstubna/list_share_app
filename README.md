## Running locally

1. Run

        $ rails s

2. Browse to <http://localhost:3000>

## Tests

1. Run tests

        $ rspec

      or run tests in a single file by

        $ rspec path_to_file

2. Run test coverage report

        $ coveralls report

    and open the `coverage/index.html` file


3. Run rubocop

        $ rubocop
